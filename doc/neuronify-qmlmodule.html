Neuronify QML Types

This is a list of all documented QML Types in Neuronify.

ACClamp

An alternating current generator which can suply input to neurons

AdaptationControl

Controls for adaptation current

AdaptationTimeConstantControl

Control for adaptation current time constant

BoundSlider

Slider control

CapacitanceControl

Control for memebrane capacitance

CurrentClamp

A direct current generator which can suply input to neurons

FileManager

A type to contain the save/load features

FireOutputControl

Control for fire output (conductance) strength in units of Siemens

InitialPotentialControl

Control for initial potential of a cell right after firing

LabelControl

Control for neuron label

MeterControls

Contains the user controls for the meters

Neuronify

This type holds the application

Note

Annotate simulations with notes

PoissonGenerator

An spike generator which can supply input spikes to neurons

RasterPlot

Shows firing times of neurons

RatePlot

Reads the firing rate of the neurons and shows a trace plot

RefractoryPeriodControl

Control for refractory period of a neuron

ResistanceControl

Control for membrane resistance

RestPotentialControl

Control for resetting potential

Retina

Visual sensor that can be connected to neurons to generate activity based on the receptive field of the sensor and the visual stimuli captured by the camera

RhythmGenerator

An spike generator which can supply input spikes to neurons

Speaker

A speaker that emits a sound whenever a connected neuron fires

SynapticOutputControl

Control for synaptic fire output (conductance) strength in units of Siemens

SynapticTimeConstantControl

Control for synaptic time constant

ThresholdControl

Control for firing threshold

TouchSensor

Registers touch (or mouse) events and converts them into a current that may be injected into neurons

Voltmeter

Reads the voltage of the neurons and shows a trace plot
